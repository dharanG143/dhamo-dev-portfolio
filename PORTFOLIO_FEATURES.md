# 🚀 Dhamodharan's Enhanced Portfolio

A modern, fully responsive portfolio website built with React.js, featuring dark/light mode support, smooth animations, and comprehensive project showcase.

## 📋 Features Implemented

### ✨ New Sections Added
- **Featured Projects** - Showcase the best projects with detailed information
- **Professional References** - Contact information for professional references with expandable cards
- **Enhanced Skills & Experience** - Full-stack, backend, tools, and soft skills sections
- **Java Projects Integration** - Electricity Billing System featured prominently
- **Resume Download** - Download as TXT or HTML format with one click

### 🌓 Dark/Light Mode
- **Toggle Button** - Fixed position button to switch between dark and light themes
- **System Preference Detection** - Automatically detects system preference on first load
- **LocalStorage Persistence** - Remembers user preference across sessions
- **Smooth Transitions** - All color changes animate smoothly

### 📧 Contact Form
- **Email Integration** - Form submissions with validation
- **Error Handling** - Real-time validation for name, email, subject, and message
- **Success Feedback** - Visual confirmation when message is sent
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop

### 📱 Responsive Design
- **Mobile First** - Optimized for all device sizes (480px - 2560px+)
- **Tablet Support** - Perfect layout on iPad and medium screens
- **Desktop Experience** - Full-featured experience on large screens
- **Touch Friendly** - All buttons and interactions work great on touch devices

### 🎨 Components
1. **Navbar** - Fixed navigation with mobile hamburger menu
2. **Hero** - Eye-catching landing section with CTA buttons
3. **About** - Personal introduction and background
4. **Skills** - Categorized skills with tab switching
5. **Projects** - Project cards with filtering and detailed information
6. **Featured** - Showcase featured projects with highlights
7. **Experience** - Work experience timeline
8. **References** - Professional references with contact info
9. **Resume** - Full resume with multiple download formats
10. **Contact** - Contact form with validation
11. **Footer** - Social links and copyright

## 📦 Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps
1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm start
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

## ⚙️ Configuration

### Resume Download
Download options available:
- **TXT Format** - Plain text resume
- **HTML Format** - Styled HTML with CSS
- **Print** - Print directly from browser

### Email Setup (Optional)
To enable email notifications for contact form submissions:
1. Sign up for [Formspree](https://formspree.io/) or similar service
2. Create a new form endpoint
3. Update the form endpoint in `Contact.jsx`

### Dark/Light Mode
The dark/light mode is automatically configured. Users can:
- Click the toggle button (sun/moon icon) to switch
- Preference is saved in browser localStorage
- System preference is detected on first load
